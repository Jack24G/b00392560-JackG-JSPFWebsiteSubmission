About Me

Hi, my name is Jack Glen, I am a JavaScript Games: Programming Fundamentals student showcasing my BabylonJS project. This project demonstrates several key aspects of 3D graphics and interactive environments. It includes the use of basic shapes, lighting, textures, shadows, and motion within a 3D scene. One scene includes a 360-degree view with textured meshes, mesh merging, and a terrain using a height map. Others feature proper lighting, camera constraints, and a movable player mesh with animations and physics interactions. The project also has a GUI with a menu system, scene transitions, and responsive elements, while audio can be toggled on for background music and sound effects that change based on in-game events.
